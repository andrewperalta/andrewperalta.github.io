A B O U T

I’m Andrew Peralta, an MBA-trained Senior Game Designer with a background in software development and a deep focus on user experience. Shrink Runner 42

Technology: Unity, C#

Platform: Web

Status: Finished

Shrink Runner 42 is a casual 3D game that involves physics, quick wits, and dexterity. The player controls the sphere and needs to collect powerups which makes up for the score and increases the play area. As the game go by, the area slowly shrinks and obstacles appear between the sphere and the powerups. It's game over when the sphere falls from the area or the area disappears completely.

  • I made this game as a compo entry for Ludum Dare 42
  • Ludum Dare game jam is a famous online game jam held twice a year.
  • Theme for Ludum Dare 42 is Running out of Space.
  • Shrink Runner 42 ranked 302nd out of 3068 in the theme division... not bad for my 1st game jam? xD
